Sistema de Encuestas

Proyecto realizado para capturar encuestas en campo. Incluye funciones como preguntas de distintos tipos, grabación de audio para verificación y captura e integración con Google Maps y GPS para designar áreas de trabajo, monitoreo de encuestadores y localización de puntos de captura. Contiene herramientas de gráficas y consultas para análisis de resultados.

Tecnología Utilizada

Se utilizó Laravel en el backend y Vue.js en el frontend. La aplicación móvil se desarrolló utilizando Android Studio (Java). La comunicación entre ambas partes se realiza con una API REST. Para la integración con mapas se utiliza la API de Google Maps y Firebase para notificaciones.

Ordena en Restaurantes Facil y Rapido

Sistema diseñado para restaurantes, su objetivo principal es digitalizar el menú del establecimiento para mostrarlo a los clientes de una manera más rápida y sencilla, además de permitirles ordenar desde el dispositivo móvil que se conectara con un sistema en la cocina para agilizar los procesos.

Tecnología Utilizada

El servidor es una aplicación Java Maven con la librería Jersey. La aplicación móvil se desarrolló utilizando Android Studio (Java). La comunicación entre ambas partes se realiza a través de una API REST.

Estudio Inteligente

Heugen es una aplicación Web diseñada para facilitar la tarea de memorizar contenidos de exámenes o conceptos. Para lograr esto se utiliza una técnica de evaluación y repetición con un algoritmo que analiza y da prioridad a los contenidos que se deben estudiar mas, en base a los resultados obtenidos

Tecnología Utilizada

Se utilizó el framework Laravel en el backend y Vue.js en el frontend. Los algoritmos de decisión fueron escritos utilizando PHP. En cuanto a la base de datos se utiliza MySQL.

Control Escolar Mejorado

Plataforma diseñada para la Facultad de Ingeniería Mecánica y Eléctrica Unidad Norte de Monclova, Coahuila México. Integra un sistema web y una aplicación móvil para controlar la asistencia de alumnos a través de códigos QR, además de manejar horarios, exámenes y otros aspectos.

Tecnología Utilizada

Se utilizó ASP.NET con C# en el backend y jQuery en el frontend. La aplicación móvil fue creada con Android Studio (Java). La comunicación entre ambas partes se realiza a través de Web Services con SOAP. Para la base de datos se utilizó SQL Server.

Plataforma de Contenidos

Sistema creado para administrar contenidos de materias o topicos de estudio. Estos contenidos se organizan en paquetes descargables, y pueden incluir examenes, preguntas, textos, etc. Ademas se incluye un chatbot que puede ser entrenado para resolver preguntas de cualquier tema.

Tecnología Utilizada

El servidor utiliza ASP.NET con C#. La aplicación móvil fue creada con Android Studio (Java). La comunicacion entre ambas partes se realiza a traves de Web Services con SOAP. Para la base de datos se utilizo SQL Server. El chatbot funciona gracias al lenguaje AIML.
