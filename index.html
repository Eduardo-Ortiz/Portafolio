Sistema de Encuestas

Proyecto realizado para capturar encuestas en campo. Incluye funciones como preguntas de distintos tipos, grabacion de audio para veritifacion y captura e ingetracion con Google Maps y GPS para designar areas de trabajo, monitoreo de encuestadores y localizacion de puntos de captura. Contiene herramientas de graficas y consultas para analisis de resultados.

Tecnologia Utilizada

Testidy utiliza ASP.NET con C# en el backend complementado con jQuery en el frontend. La aplicación movil fue creada con Android Studio (Java). La comunicacion entre ambas partes se realiza a traves de Web Services con SOAP. Para la base de datos se utilizo SQL Server.

Estudio Inteligente

Heugen es una aplicación Web diseñada para facilitar la tarea de memorizar contenidos de examenes o conceptos. Para lograr esto se utiliza una tecnica de evaluacion y repeticion con un algoritmo que analiza y da prioridad a los contenidos que se deben etudiar mas, en base a los resultados obtenidos.

Tecnologia Utilizada

Se utilizo el framework Laravel en el backend y Vue.js en el frontend. Los algoritmos de decision fueron escritos utilizando PHP. En cuanto a la base de datos se utiliza MySQL.
