Estudio Inteligente

Heugen es un proyecto personal creado para facilitar las tareas de aprendizaje y memorización a través de un sistema inteligente que mantiene un control de lo que se debe estudiar en mayor medida dentro de un grupo de preguntas.

Para esto se utiliza una metodología de estudio y repetición para memorización de conceptos, tomando en cuenta la velocidad de lectura del usuario para lograr la mayor eficiencia y su efectividad para priorizar los conceptos que requieren más atención.

  • aplicación web responsiva, con funciones como:
    • registro de cuentas y login;
    • test para medir la velocidad de lectura y adaptar los tiempos de estudio al ritmo único de cada usuario;
    • creación de pools de preguntas para estudiar, con una amplia variedad de opciones para la personalización del mismo a las necesidades del usuario;
    • función para explorar pools creados por otros usuarios e inscribirse a estos;
    • función de estudio con un algoritmo que elige preguntas al azar del pool, pero a la vez da prioridad a las preguntas con más errores;
    • función para resaltar las palabras clave de cada pregunta en caso de que el usuario comenta muchos errores en el mismo punto;
    • estadísticas de efectividad y tiempo de estudio al terminar de memorizar un pool de preguntas;
    • comparación con las estadísticas y resultados obtenidos por otros usuarios para medir grados de efectividad;
Explicación Técnica
Aplicación Web
  • Aplicación Web Laravel.
  • Utilización de Vue.js en el front end para distintas aplicaciones.
  • Algoritmos de aleatoriedad y prioridad para estudio escritos utilizando PHP.
  • Base de datos en MySQL.
  • Bulma utilizado como framework CSS para diseño y hacer la aplicación responsiva.
